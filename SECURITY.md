# Security Policy

## Supported Versions

The following table outlines which versions of **@travjek/apig** currently receive security updates.

| Version | Supported |
|---------|-----------|
| 0.8.x (latest) | :white_check_mark: |
| < 0.8.0 | :x: |

> **Note:** Security fixes are applied only to supported versions. Users are strongly encouraged to upgrade to the latest release whenever possible.

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public GitHub issue.**

Report it privately via one of these channels:

- **GitHub Security Advisories:** [Report a vulnerability](https://github.com/elPyzi/apig/security/advisories/new)
- **Email:** danila.pyzhikov@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect a response within **72 hours**. Once confirmed, we will release a patch as soon as possible and credit you in the release notes (unless you prefer to stay anonymous).
